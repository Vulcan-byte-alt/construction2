# Project: Roca Group Website
## Client: Roca Group — construction firm
## Goal: Premium WordPress theme, dark/charcoal palette, amber/gold accents matching logo
## Logo colors: charcoal #1a1a18, gold/amber #c8922a
## Design reference: Dark hero with diagonal bottom cut, ticker banner below hero
## Stack: PHP (WordPress), Tailwind CSS (CDN for dev, compiled for prod), Vanilla JS, GSAP (CDN)
## Pages: Home, Our Approach, Our Sectors, Our Projects (Current + Completed), Contact Us
## WordPress features needed: ACF for Projects CPT, WP Menus, Contact Form 7

## Coding Standards
- Mobile-first CSS
- Semantic HTML5
- No jQuery (vanilla JS only unless WP forces it)
- GSAP for scroll animations
- DO NOT touch functions.php without listing what you're adding and why

## DO NOT
- Use page builders
- Use inline styles except for dynamic JS values
- Hardcode any content that should be editable in WP dashboard
